# LLM Primer

**系统学习大语言模型：从原理到实践**

一个双语、分层次、社区驱动的 LLM 学习知识库，涵盖基础概念、工程实践、前沿论文与可交互演示。

## 这是什么 / 给谁 / 要多久

- **30 分钟**：通过直觉版建立心智模型，理解 LLM 核心概念
- **3 小时**：完成工程档，掌握落地实践与性能优化
- **按需深入**：在研究档追踪论文脉络、开放问题与前沿进展
