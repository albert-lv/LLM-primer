# 为什么要做 LLM Primer

LLM 发展太快，很多资料要么只讲直觉，要么直接跳到论文细节。LLM Primer 希望把两者连接起来：每篇文章都提供直觉版、工程版，必要时再加研究版，让读者可以按自己的背景选择深度。

这个项目也把论文当作可追溯证据。文章中的关键说法应尽量连接到已收录论文；尚未核验的候选论文先放进 inbox，避免把未经整理的引用混入正文。

我们欢迎社区贡献：新增文章、补充英文翻译、改进互动 demo、认领论文 TLDR 或修正错误。目标不是追热点，而是沉淀一套能长期维护的 LLM 学习地图。
